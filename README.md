# REQUIS
- node js

# INSTALLATIONS

[^1]: Ouvrir un terminal et utiliser la commande suivante :

```bash
git clone https://github.com/H4CK3R5-Dmaster/toxaakbot
cd toxaakbot
npm install
```

[^2]: Créer un fichier configs.json sous la structure suivante :

```json
{
    "token":"token bot"
}
```

[^3]: Pour démarrer :

```bash
node .
```
